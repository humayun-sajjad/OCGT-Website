# company_logos/

Tool-vendor logos used in the "Our Toolchain" strip on the Reality Capture
overview page (`#p-rc`).

## Expected filenames

Drop an SVG (preferred) or PNG with transparent background here with the exact
filename below. The site picks it up automatically — no code changes needed.

- `pix4d.svg`
- `agisoft.svg`          (Agisoft Metashape)
- `realitycapture.svg`   (Epic / Capturing Reality)
- `dji-terra.svg`
- `zephyr.svg`           (3DF Zephyr)
- `3dsurvey.svg`
- `cloudcompare.svg`
- `qgis.svg`
- `autocad.svg`          (AutoCAD Civil 3D)
- `revit.svg`

Until the file is present, a styled wordmark is shown in its place. Both
states use the same tile dimensions and monochrome treatment so layout is
stable.

## Legal note

Only use official press-kit / brand-assets logos in the nominative
fair-use sense ("software we use"). Do **not** add client/project-owner
logos here without written permission.
